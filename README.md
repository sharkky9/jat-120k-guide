# Julian Alps Trail 120K Race-Day Guide

A static, evidence-backed race-day field guide for planning the Julian Alps Trail 120K.

## Site

GitHub Pages source: `docs/`

Local preview:

```bash
python3 -m http.server 4173 --directory docs
```

Then open `http://localhost:4173/`.

## Included public artifacts

- `docs/index.html` - main guide
- `docs/assets/data/course-data.js` - generated course/evidence data bundle
- `docs/assets/images/` - visual assets used by the guide
- `docs/research/` - dossier and CSV evidence exports

The guide intentionally labels modeled surfaces and evidence caveats. Surface totals are model outputs, not official race declarations.
